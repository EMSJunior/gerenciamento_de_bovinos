# Módulo de Login

Responsável: Thiago Tavares Santos Vasconcelos

Este módulo será responsável pela autenticação dos usuários do sistema de gerenciamento de bovinos.

## Objetivo

Permitir que diferentes tipos de usuários acessem o sistema de acordo com sua função.

## Tipos de usuários previstos

- Gerente
- Veterinário
- Peão

## Funcionalidades planejadas

- Validar nome de usuário e senha.
- Identificar o tipo de usuário autenticado.
- Controlar o acesso às funcionalidades de acordo com o perfil do usuário.
